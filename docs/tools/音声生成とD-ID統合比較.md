## ⚠️ 主な懸念点・弱点

### Cartesia + D-ID の懸念点

| 懸念点 | 詳細 | 対処法 |
|-------|------|--------|
| **やや早口** | 通常より若干速い話し方 | 句読点を多めに入れる<br>改行で間を作る |
| **長文での違和感** | 5分以上の長文で違和感が残る場合 | スクリプトを短文に分割<br>テストで確認 |
| **感情表現やや劣る** | ElevenLabsと比較して | 重要動画のみElevenLabsへ |
| **API統合必須** | Webスタジオなし | 開発チーム必要 |
| **タイムスタンプ非対応** | 日本語でのタイムスタンプ機能なし | - |

### ElevenLabs + D-ID の懸念点

| 懸念点 | 詳細 | 対処法 |
|-------|------|--------|
| **コスト高** | 年間+¥30,000 | 予算に余裕がある場合のみ |
| **サンプル時間長** | 1-2分の録音必要 | - |
