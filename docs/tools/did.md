## 📊 価格体系

### Lite Plan
- **月払い**: 926円/月
- **年払い**: 727円/月（年間8,724円）
- **動画分数**: 月10分
- **最低エントリー価格**でテスト可能

### Pro Plan
- **月払い**: 7,730円/月
- **年払い**: 2,475円/月（年間29,700円）
- **動画分数**: 月15分
- **音声クローニング**: 1つ

### Advanced Plan
- **月払い**: 46,391円/月
- **年払い**: 16,708円/月（年間200,496円）
- **動画分数**: 月65分
- **音声クローニング**: 3つ
