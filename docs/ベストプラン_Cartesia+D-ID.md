### Day 2: 音声クローン + D-ID

#### 4. 音声クローン作成（5分）

**Python実装例:**
```python
import requests
import os

CARTESIA_API_KEY = "your_api_key_here"

def create_voice_clone(audio_file_path, voice_name="MyVoice"):
    url = "https://api.cartesia.ai/voices/clone"

    headers = {
        "Authorization": f"Bearer {CARTESIA_API_KEY}",
        "Cartesia-Version": "2025-04-16"
    }

    with open(audio_file_path, "rb") as audio_file:
        files = {"clip": audio_file}
        data = {
            "name": voice_name,
            "language": "ja"
        }
        response = requests.post(url, headers=headers, files=files, data=data)

    if response.status_code == 200:
        result = response.json()
        print(f"✅ 音声クローン作成成功！")
        print(f"Voice ID: {result['id']}")
        return result['id']
    else:
        print(f"❌ エラー: {response.status_code}")
        return None

# 実行
voice_id = create_voice_clone("my_voice_sample.mp3")
```

**Voice IDを保存:**
```bash
# .env ファイルに保存
CARTESIA_API_KEY=your_api_key_here

```
