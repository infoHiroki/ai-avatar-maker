## ステップバイステップ実装

### ステップ1: アカウント作成（5分）

```bash
1. https://studio.d-id.com にアクセス

2. 「Sign Up」をクリック

3. 登録方法を選択:
   - Googleアカウント（推奨）
   - Emailアドレス

4. メール認証を完了

5. ダッシュボードにログイン
```

---

### ステップ2: 最初のテスト動画作成（10分）

#### 方法A: 既存アバターを使う（最速）

```bash
1. 「Create Video」をクリック

2. 「Presenters」タブから好きなアバターを選択
   推奨: 日本人風の自然な顔

3. 「Script」タブに移動

4. テキスト入力:
   「こんにちは。今日はAIアバターについてお話しします。」

5. 言語設定: 「Japanese」を選択

6. 声の選択:
   - 男性/女性
   - トーン（落ち着いた/明るい）

7. 「Generate Video」をクリック

8. 1-2分待つ

9. 完成したら「Download」
```

#### 方法B: 自分の写真を使う（推奨）

```bash
1. 自分の顔写真を用意:
   - 正面を向いている
   - 明るい場所で撮影
   - 背景はシンプル
   - サイズ: 512x512以上

2. 「Create Video」→「Upload Image」

3. 写真をアップロード

4. 上記と同じ手順でスクリプト入力

5. 生成・ダウンロード
```

**所要時間**: 10分
**コスト**: 無料枠内

---

### ステップ3: ブログ記事から動画作成（20分）

#### 3-1. ブログ記事をYouTubeスクリプトに変換

```bash
1. ChatGPTを開く（https://chat.openai.com）

2. 以下のプロンプトを使用:

---プロンプト開始---
あなたはブログ記事をYouTube動画スクリプトに変換する専門家です。

【変換ルール】
- 尺: 5分（話し言葉で1000文字）
- 構成:
  * オープニング（15秒）: フック
  * 本編（4分）: 3つのポイント
  * クロージング（15秒）: まとめ + CTA
- 話し方: です・ます調、親しみやすく

【元記事】
[ここにブログ記事のURLまたは本文を貼り付け]

YouTubeスクリプトに変換してください。
---プロンプト終了---

3. 生成されたスクリプトをコピー
```

#### 3-2. D-IDで音声付き動画生成

```bash
1. D-ID Studioに戻る

2. 「Create Video」

3. アバターを選択（自分の写真 or 既存）

4. スクリプトを貼り付け

5. 「Generate Video」

6. 3-5分待つ

7. ダウンロード
```

**所要時間**: 20分
**コスト**: 無料枠内（5分まで）

---

### ステップ4: YouTube投稿（10分）

```bash
1. YouTubeにログイン

2. 「作成」→「動画をアップロード」

3. D-IDで生成した動画を選択

4. タイトル・説明文を入力:
   - タイトル: ブログ記事タイトルを参考に
   - 説明文: ブログ記事のURLも記載

5. サムネイル設定:
   - 自動生成 or Canvaで作成

6. 「公開」or「限定公開」で投稿

7. ブログ記事に動画を埋め込み
```

**初回動画完成！🎉**

## 品質向上のコツ

### アバター画像の準備

```
推奨仕様:
  解像度: 1024x1024以上
  形式: JPG, PNG
  背景: 単色またはぼかし
  表情: 自然な笑顔
  服装: ビジネスカジュアル
  照明: 均一で明るい
```

### 音声の最適化

```
D-ID標準音声の場合:
  - 速度調整: 0.9-1.1倍（デフォルト1.0）
  - ピッチ: 自然な範囲で調整
  - 間を取る: 「、」や「。」で自動的に間が入る

カスタム音声を使う場合:
  - 事前にElevenLabsやOpenAI TTSで生成
  - 音声ファイルをアップロード
  - より自然な発音が可能
```

### スクリプトの工夫

```
効果的なスクリプト:
  ✅ 短い文章（1文20文字以内）
  ✅ 読点を適切に入れる
  ✅ 難しい専門用語を避ける
  ✅ 数字は漢数字より算用数字（「3つ」>「三つ」）

避けるべき:
  ❌ 長すぎる文章
  ❌ 読点なしの文章
  ❌ 外来語の多用
  ❌ 早口言葉的な表現
```

## よくあるトラブル

### Q1: 音声が不自然

**原因**: 日本語の音声エンジン選択ミス
**解決**:
```
推奨音声:
  - Microsoft: ja-JP-NanamiNeural（女性）
  - Microsoft: ja-JP-KeitaNeural（男性）
  - Google: ja-JP-Neural2-A
```

### Q2: リップシンクがずれる

**原因**: 音声の長さとスクリプトの不一致
**解決**:
- スクリプトを短くする
- 音声速度を調整
- 一文ずつ区切って生成

### Q3: 無料枠がすぐなくなる

**原因**: 失敗した動画もカウントされる
**解決**:
- 最初は短いテキストでテスト
- スクリプトを事前に確認
- Liteプラン($5.9)にアップグレード
