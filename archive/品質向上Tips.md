## ✍️ スクリプト作成の5つのコツ

### 1. 明確な発音

**NG**:
```
つぶやきや複雑な早口言葉
「歯周病治療においてプロービングデプスとアタッチメントレベルの測定が重要」
```

**OK**:
```
短く明確に
「歯周病の診断では、
プロービングデプスと
アタッチメントレベルの測定が
重要な指標となります。」
```

---

### 2. 自然なペース

**NG**（堅い文書調）:
```
本日は、歯周病の初期診断における
重要な指標について解説いたします。
```

**OK**（会話調）:
```
今日は、歯周病の初期診断について
お話しします。
```

**ポイント**:
- 1文は短めに（20-30文字程度）
- 「です・ます」調で統一
- 自然な会話のリズム
- 難しい表現を避ける

---

### 3. 感情的なキュー

**句読点の活用**:
```
歯周病の診断では、【、← ここで一呼吸】
プロービングデプスと【、← ここで一呼吸】
アタッチメントレベルが重要です。【。← ここで完全な休止】
```

**強調マーカー**:
```
**特に重要なのは**、
初期段階での診断です。
```

**ポイント**:
- 句読点で自然な間を作る
- 強調したい部分は短い文に
- 疑問形で視聴者を引き込む

---

### 4. 最適な長さ

**推奨**:
- 1本の動画: **3〜5分**
- 1分 = 約300-350文字
- 3分 = 約900-1,050文字

**構成例（3分動画）**:
```
導入（30秒、約100字）
├─ 挨拶と今日のテーマ
├─ なぜ重要か

本編（2分、約700字）
├─ ポイント1（40秒、約230字）
├─ ポイント2（40秒、約230字）
├─ ポイント3（40秒、約230字）

まとめ（30秒、約100字）
├─ 要点の復習
├─ 次回予告
└─ CTA（チャンネル登録など）
```

**ポイント**:
- 3-5分で視聴者のエンゲージメント維持
- 長すぎると離脱率が高まる
- 短すぎると価値提供が不十分

---

### 5. 字幕必須

**理由**:
- アクセシビリティ向上
- 音声オフ視聴（通勤中など）
- 専門用語の確認
- SEO効果

**方法**:
- YouTube自動字幕生成
- 手動で編集・修正
- 専門用語は正確に

## 🎤 音声品質向上

### 高品質マイク使用

**推奨マイク**:
- USB接続マイク: 5,000-10,000円
- Blue Yeti、Audio-Technica AT2020USB+等

**効果**:
- クリーンな音声 = より良いリップシンク
- AIアバターの音声クローン品質向上
- プロフェッショナルな印象

---

### 録音環境

**理想的な環境**:
- 🔇 静かな場所
- 🚫 エアコン音を最小限に
- 🚫 PC音も最小限に
- 📏 マイクから15-20cmの距離

**NG環境**:
- ❌ カフェなど騒音のある場所
- ❌ エアコン直下
- ❌ PCファンの音が大きい場所
- ❌ エコーする部屋

---

## 📷 カスタムアバター撮影

### 顔の位置

**推奨**:
- 顔を画面中央に配置
- 正面を向く
- 自然な表情
- 録画中の手振りを最小限に

**NG**:
- ❌ 顔が左右に偏っている
- ❌ 上を向いている/下を向いている
- ❌ 過度な身振り手振り
- ❌ 表情が固い

---

### 照明

**推奨**:
- 均一で自然な照明
- 顔に影を作らない
- 柔らかい光
- 窓際の自然光（理想）

**NG**:
- ❌ 逆光
- ❌ 強い影
- ❌ 暗すぎる
- ❌ 蛍光灯の直射

**簡単な照明セットアップ**:
```
    ライト1
      ↓
   👤（自分）
      ↓
    カメラ
```

または3点照明:
```
ライト1  ライト2
   ↓  ↓
    👤
    ↓
   カメラ
    ↑
  ライト3
（背景用）
```

2. 異なる背景を試す
3. 異なるアバターを試す

```

## 📈 継続的改善

### 視聴者フィードバックの活用

- YouTube コメント

```

## 🏆 成功事例の分析

### 人気の歯科教育動画の特徴

1. **明確なタイトル**
   - 「〇〇の3つのステップ」
   - 「初心者向け〇〇ガイド」
   - 「よくある〇〇の間違い」

2. **構造化された内容**
   - 導入・本編・まとめ
   - 番号付きポイント
   - 視覚的な図表

3. **適切な長さ**
   - 3-5分が最適
   - 長すぎず短すぎず

4. **高品質な音声・映像**
   - クリアな音声
   - プロフェッショナルな見た目
   - 適切な照明

## 結論

**品質向上の優先順位**:

1. **スクリプト**（最重要）
   - 会話調
   - 短い文
   - 適切な句読点

2. **音声**
   - 高品質マイク
   - 静かな環境
   - 明確な発音

3. **リップシンク**
   - 適切なツール選択
   - スクリプト調整

4. **視覚**
   - 適切な背景
   - 良い照明
   - プロフェッショナルなテンプレート

5. **継続的改善**
   - フィードバック収集
   - データ分析
   - 次回動画に反映

**これらを守れば、プロフェッショナルな品質の歯科教育動画を制作できます。**
